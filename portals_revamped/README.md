# Revamped

This is a collection of redesigned captive portals based on the most popular services. The goal is to create a more realistic and up-to-date experience for users who encounter these portals, while also improving the security and functionality of the pages.

## Features

- Modern and responsive design that mimics the original portals closely
- Improved form validation and error handling
- Works completely offline, with all assets included in the repository
- Tested with Flipper Zero's Evil Portal app for compatibility

## Revamped vs Revamped Plus

### Revamped

- Faithful replication of original captive portals
- Modern design updates

### Revamped Plus (Recommended)

Same modern design as Revamped, but with extra features and improvements:

- Clean captive portal start page
- Realistic landing page experience
- Clear call-to-action and seamless transitions to login pages
- Gzip compressed for reduced file size
